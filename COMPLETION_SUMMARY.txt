🚀 كيفية البدء:

1️⃣ تثبيت المكتبات:
   pip install -r requirements.txt

2️⃣ اختبار التثبيت:
   python test_installation.py

3️⃣ إضافة ملف الدردشة:
   - صدّر دردشة من WhatsApp
   - احفظها باسم chat.txt في نفس المجلد

4️⃣ بدء الخادم:
   python main.py
   ثم اذهب إلى: http://127.0.0.1:8000

5️⃣ أو حلل محلياً:
   python analyze_chat.py
