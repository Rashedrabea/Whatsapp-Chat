═══════════════════════════════════════════════════════════════════

✨ التطبيق الآن يعمل على الويب (Web)!

يمكنك فتحه على:
✅ كمبيوتر (Desktop)
✅ موبايل (Mobile)
✅ أي جهاز متصل بالإنترنت

═══════════════════════════════════════════════════════════════════

🚀 الخطوات:

1️⃣ فتح Terminal/PowerShell

2️⃣ الذهاب إلى مجلد المشروع:
   cd C:\Users\dell\Desktop\Whatsapp-Chat-Analyzer-main

3️⃣ تشغيل الخادم:
   python main.py

   ستظهر رسالة مثل:
   INFO:     Uvicorn running on http://127.0.0.1:8000

4️⃣ فتح المتصفح:

   على الكمبيوتر:
   http://127.0.0.1:8000

   على الموبايل (نفس الشبكة):
   1. اعرف IP الكمبيوتر:
      ipconfig
   2. ابحث عن "IPv4 Address" (مثال: 192.168.1.100)
   3. افتح في الموبايل:
      http://192.168.1.100:8000

═══════════════════════════════════════════════════════════════════

💻 على الكمبيوتر:

1. اتبع الخطوات أعلاه
2. افتح المتصفح
3. اكتب: http://127.0.0.1:8000
4. ستظهر الواجهة الجميلة! 🎨

═══════════════════════════════════════════════════════════════════

📱 على الموبايل:

الطريقة 1: على نفس الشبكة (WiFi):

1. تأكد أن الموبايل والكمبيوتر على نفس الشبكة
2. افتح Terminal على الكمبيوتر:
   python main.py

3. اعرف IP الكمبيوتر:
   ipconfig
   ابحث عن: IPv4 Address (مثل 192.168.x.x)

4. في موبايلك افتح المتصفح واكتب:
   http://192.168.x.x:8000

5. ستظهر الواجهة الكاملة! ✨

الطريقة 2: على نفس الجهاز:

إذا كان لديك متصفح على الكمبيوتر:
http://127.0.0.1:8000

═══════════════════════════════════════════════════════════════════

🔍 كيف تعرف IP الكمبيوتر:

على Windows PowerShell:
1. اكتب: ipconfig
2. ابحث عن "IPv4 Address"
3. ستجد شيء مثل: 192.168.1.100
4. استخدم هذا الرقم: http://192.168.1.100:8000
